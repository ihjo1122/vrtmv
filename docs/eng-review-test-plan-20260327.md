## Affected Pages/Routes
- MainScreen — 모델별 AR Camera 버튼 (INT4/INT8), 미다운로드 모델 클릭 시 다운로드 다이얼로그
- CameraScreen — 터치 → 검출 → 바운딩박스 표시, VLM 추론 + 시간 표시, 모델명 표시
- IntroScreen — 기본 모델 자동 다운로드 (변경 없음, 기존 동작 확인)

## Key Interactions to Verify
- 터치한 위치에 바운딩박스가 정확히 맞는지 (좌표 매핑 수정 후)
- 모델별 버튼 클릭 → 올바른 모델로 CameraScreen 진입
- 미다운로드 모델 클릭 → 다운로드 다이얼로그 → 완료 → CameraScreen 진입
- VLM ON 상태에서 추론 시간(ms)이 AR 오버레이에 표시
- 모델 전환: CameraScreen → 뒤로 → MainScreen → 다른 모델 선택

## Edge Cases
- 저장공간 부족 시 다운로드 시도 → 에러 메시지
- 모델 로드 실패 → MainScreen 복귀
- 객체가 하나도 검출되지 않았을 때 터치 → "검출 없음" 표시
- 검출된 객체가 화면 가장자리에 있을 때 바운딩박스 클리핑
- 다운로드 중 앱 백그라운드 → 포그라운드 복귀 시 진행률 유지

## Critical Paths
- 터치 → 검출 → 바운딩박스 정확히 맞음 (핵심 시연 경로)
- MainScreen → 모델 선택 → CameraScreen → VLM 추론 → AR 표시 (전체 E2E)
- 모델 전환 경로: CameraScreen A → 뒤로 → MainScreen → CameraScreen B (메모리 해제 확인)
